#### How I think about open source

Default stance: open. If a project saves people time but they could build it themselves, it should be free. AI-assisted development means everyone can execute at the speed of thought — locking progress behind a paywall just invites someone to replicate and undercut you.

When something becomes a hosted service, pricing is cost-plus — cover infrastructure, add a modest margin. Self-host if you want. Pay for convenience if you don't.
